# 20260120-RKE2-클러스터-기반-구축-및-ACM-연동

## 1. 개요
핵심 워크로드 실행 환경인 RKE2 클러스터를 AWS EC2 기반으로 구축하고, 외부 노출을 위한 NLB 및 ACM TLS 환경을 구성함.

## 2. 주요 작업 내용
- **RKE2 컨트롤 플레인 배포**: 3-Node 고가용성 컨트롤 플레인 구성을 위해 User Data 기반 자동 설치 스크립트 작성.
- **ACM Certificate 연동**: Public 도메인(unifiedmeta.net) 기반의 ACM 인증서를 NLB(Network Load Balancer)에 태깅하여 HTTPS 통신 기반 마련.
- **내부 로드밸런싱**: Supervisor 및 API Server를 위한 Internal NLB 구성으로 노드 간 통신 안정성 확보.

## 3. 결과 및 가치
- 운영 가능한 수준의 첫 쿠버네티스 클러스터 인프라 완성.
- 외부 공용 DNS와 연계된 안전한 서비스 접속 환경 구축.
