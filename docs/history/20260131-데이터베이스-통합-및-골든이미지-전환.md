# 20260131-데이터베이스-통합-및-골든이미지-전환

## 1. 개요
분산되어 있던 DB 스택을 하나로 통합하고, 보안 및 성능이 검증된 커스텀 골든 이미지(Golden Image) 기반으로 인프라를 현대화함.

## 2. 주요 작업 내용
- **60-db 스택 통합**: PostgreSQL, Neo4j 등 여러 DB 리소스를 단일 `60-db` 스택으로 병합하여 관리 포인트 일원화.
- **골든 이미지 적용**: OS 경량화 및 보안 설정이 완료된 사내 표준 골든 이미지를 AMI로 사용하도록 EC2 설정 업데이트.
- **고가용성 확보**: 가용영역(AZ) 분산 배치를 통해 단일 장애점(SPOF)을 제거하고 서비스 안정성 강화.

## 3. 결과 및 가치
- 인벤토리 관리 복잡도 감소.
- 보안 컴플라이언스가 준수된 균일한 서버 환경 운영 가능.
