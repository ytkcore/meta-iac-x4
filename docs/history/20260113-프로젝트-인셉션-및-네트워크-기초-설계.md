# 20260113-프로젝트-인셉션-및-네트워크-기초-설계

## 1. 개요
전사 공통 인프라 플랫폼 구축을 위한 프로젝트의 시작 단계로, 퍼블릭 클라우드(AWS) 기반의 확장 가능한 네트워크 아키텍처를 설계하고 기초 코드를 작성함.

## 2. 주요 작업 내용
- **네트워크 레이아웃 정의**: VPC CIDR(10.0.0.0/16) 설정 및 가용영역(AZ)별 Public/Private 서브넷 티어링 구조 확립.
- **모듈화 아키텍처 수립**: 재사용 가능한 `modules/vpc`와 `modules/security-groups`의 초안을 작성하여 스택 간 일관성 확보.
- **태깅 규정(Standard Tags) 도입**: `ManagedBy`, `Project`, `Environment` 등 관리 자동화를 위한 공통 태그 세셋 정의.

## 3. 결과 및 가치
- 프로젝트를 위한 클린한 AWS 계정 기반 마련.
- 코드 재사용성을 극대화한 모듈 중심 개발 문화의 기틀 확산.
