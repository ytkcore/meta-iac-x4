# 20260202-인프라-긴급-장애-복구-및-백엔드-정상화

## 1. 개요
테라폼 백엔드 구성 오류와 K8s 리소스 삭제 시 발생하는 웹훅(Webhook) 차단 문제를 해결하여 인프라 형상 관리 기능을 복구함.

## 2. 주요 작업 내용
- **백엔드 중복 제거**: 여러 파일에 중복 선언되어 충돌을 일으키던 `terraform.backend` 설정을 정리하고 권장 레이아웃으로 복구.
- **K8s Finalizer 제거**: 삭제 대기 상태에서 멈춰있던 네임스페이스와 리소스들의 Finalizer를 수동 제거(`kubectl patch`)하여 리소스 고립 해결.
- **삭제 프로세스 복원**: 웹훅 서버 부재로 인한 삭제 명령 거부 현상을 해결하여 정상적인 `make destroy`가 가능하도록 조치.

## 3. 결과 및 가치
- 중단되었던 인프라 배포/삭제 파이프라인 정상 가동.
- 유력한 기술 부채였던 K8s 고립 자원 전수 정리.
