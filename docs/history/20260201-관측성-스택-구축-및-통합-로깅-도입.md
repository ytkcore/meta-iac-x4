# 20260201-관측성-스택-구축-및-통합-로깅-도입

## 1. 개요
클러스터 운영 가시성을 확보하기 위해 Longhorn 스토리지 및 Prometheus/Grafana 모니터링 시스템을 `70-observability` 전용 스택으로 구축함.

## 2. 주요 작업 내용
- **70-observability 스택 신설**: 클라우드 리소스(S3, IAM)와 K8s 리소스를 분리하여 관리 효율성 증대.
- **Longhorn 백업 자동화**: S3 백업 전용 IAM Role을 생성하고 노드에 권한을 자동 할당함.
- **중앙 집중형 로깅**: `common/logging.sh` 유틸리티를 제작하여 모든 프로젝트 스크립트에 일관된 로깅 형식(Header, Ok, Err 등)을 적용.

## 3. 결과 및 가치
- 인프라 작업 이력의 가시성을 확보하여 사후 분석 용이성 증대.
- 고가용성 스토리지 및 실시간 메트릭 대시보드 기반 마련.
