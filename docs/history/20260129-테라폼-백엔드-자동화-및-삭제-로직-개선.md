# 20260129-테라폼-백엔드-자동화-및-삭제-로직-개선

## 1. 개요
초기 인프라 설정의 진입 장벽을 낮추기 위해 S3 백엔드 생성을 자동화하고, 전체 삭제 과정의 예외 상황 처리를 강화함.

## 2. 주요 작업 내용
- **백엔드 부트스트랩 통합**: `make init` 실행 시 S3 버킷 존재 여부를 체크하여 자동으로 생성하는 로직을 Makefile에 통합.
- **삭제 로직 고도화 (`destroy-all`)**: 
    - 이미 삭제된 스택은 건너뛰도록 개선하여 전체 삭제 안정성 확보.
    - 프로세스 마지막 단계에서 S3 백엔드 버킷까지 일관되게 삭제하도록 처리.
- **변수 최적화**: `env.tfvars`에 중복된 변수들을 제거하고 모듈 내 `locals`와 `default` 값을 활용하여 설정 파일 경량화.

## 3. 결과 및 가치
- 신규 개발자의 프로젝트 투입(On-boarding) 시간 단축.
- 리소스 잔적 없는 깨끗한 클라우드 환경 정리(Cleanup) 보장.
