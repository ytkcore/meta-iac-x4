# 20260125-Harbor-레지스트리-구축-및-OCI-이미지-관리

## 1. 개요
사내 이미지 관리 및 보안 취약점 점검을 위해 Harbor를 전용 인스턴스에 구축하고, OCI 레지스트리 기능을 활성화함.

## 2. 주요 작업 내용
- **Harbor 스택 독립화**: `40-harbor` 스택을 통해 ALB, EC2 리소스를 개별 관리할 수 있도록 구성.
- **데이터 영속성 확보**: Docker 볼륨 대신 S3 백엔드를 Harbor 스토리지로 연동하여 무중단 확장성 확보.
- **OCI 프록시 캐시 설정**: 외부 공개 이미지(Docker Hub 등)의 로컬 캐싱을 통해 대역폭 비용 절감 및 배포 속도 개선.

## 3. 결과 및 가치
- 중앙 집중형 컨테이너 이미지 거점 확보.
- 보안 취약점 스캐닝을 통한 배포 전 안심 환경 구축.
