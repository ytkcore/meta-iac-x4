# 20260114-테라폼-운영-표준화-및-메이크파일-통합

## 1. 개요
멀티 스택(VPC, Security, K8s 등) 환경에서 발생하는 운영 복잡도를 해결하고, 실행 명령어를 표준화하기 위해 `Makefile` 기반의 워크플로우를 도입함.

## 2. 주요 작업 내용
- **Makefile 기반 인터페이스 구축**: `make init`, `make plan`, `make apply` 등 모든 테라폼 명령어를 환경(ENV)과 스택(STACK) 변수 기반으로 추상화.
- **멀티 테넌시(Multi-tenancy) 지원**: 동일한 코드로 여러 환경(dev, stg, prod)을 안전하게 관리할 수 있도록 `env.tfvars` 처리 로직을 Makefile에 통합.
- **환경 변수 유효성 검사**: 필수 변수 누락 시 실행을 차단하는 가드(Guard) 로직을 추가하여 휴먼 에러 방지.

## 3. 결과 및 가치
- 복잡한 테라폼 CLI 옵션을 외울 필요 없는 직관적인 운영 환경 제공.
- 환경 간 설정 실수(예: dev 설정을 prod에 적용) 가능성 원천 차단.
